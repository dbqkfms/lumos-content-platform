## 1. Ideal Customer Profile

### 1차 핵심 고객군
- 프리미엄 호텔 / 로비 운영자
  - 목적: 첫인상 강화, 체류 경험 향상, 공간의 고급감 유지
  - 중요 요소: 공간 톤, 반복 재생 피로도, 설치 후 완성도
- 리테일 / 쇼룸 / 브랜드 공간 운영자
  - 목적: 브랜드 인상 강화, 방문 동선 유도, 촬영 포인트 생성
  - 중요 요소: 시선 집중, 계절/프로모션 대응, 빠른 교체 가능성
- 전시 / 행사 / 공공형 공간 운영자
  - 목적: 강한 상징성, 현장 인상, 대형 화면 활용 극대화
  - 중요 요소: 랜드마크성, 서사, 설치 시뮬레이션, 납품 안정성

### 좋지 않은 고객군
- 즉시 결제형 소비자 아트 구독 사용자
- 텍스트 중심 광고 송출만 필요한 전통 사이니지 고객
- 커스텀 없이 초저가 템플릿만 찾는 수요

## 2. Value Proposition Statements

### 호텔 / 프리미엄 로비
LUMOS는 프리미엄 호텔과 로비 운영자가 설치 이후 비어 있는 화면을 고급스럽게 완성할 수 있도록, 공간 톤에 맞는 큐레이션과 설치 시뮬레이션을 함께 제공합니다.

### 리테일 / 쇼룸
LUMOS는 브랜드 공간과 쇼룸이 LED 하드웨어만이 아니라 실제로 “무엇을 틀 것인가”까지 제안할 수 있도록, STANDARD와 LOCAL로 나뉜 미디어아트 컬렉션과 적용 시나리오를 제공합니다.

### 전시 / 랜드마크 / 공공
LUMOS는 전시와 대형 공간 프로젝트에서 강한 존재감이 필요한 운영자를 위해, 상징성과 스케일이 살아 있는 콘텐츠와 제안용 목업을 빠르게 구성할 수 있게 돕습니다.

## 3. Customer Journey

| 단계 | 핵심 접점 | 사용자 행동 | 대표 감정 | 주요 마찰 | 기회 |
|---|---|---|---|---|---|
| Awareness | THISGLOBAL 사이트, 소개자료, 레퍼런스 | LED 공급사 탐색 | 관심, 막연함 | 콘텐츠 차별점이 약함 | LUMOS 연결 강조 |
| Consideration | LUMOS 홈, STANDARD / LOCAL, Process | 공간에 맞는 화면 상상 | 기대, 비교 | 무엇이 맞는지 어려움 | 상황 검색, 태그, 목업 |
| Inquiry | 문의 폼, 작품 상세 CTA | 공간 조건 전달 | 신중함 | 무엇을 써야 할지 모름 | 문의 유형 라우팅 |
| Proposal | 설치 시뮬레이션, 큐레이션 제안 | 내부 검토, 의사결정 | 설득 필요 | 실제 적용 확신 부족 | mockup / case 강화 |
| Delivery | 납품형 콘텐츠 세트 | 화면 반영 | 안도, 기대 | 포맷/운영 범위 불명확 | 운영지원 문구 |
| Retention | 업데이트 / 수정 요청 | 시즌/공간별 추가 요청 | 만족, 재구매 | CS 창구 불명확 | 운영지원 / CMS 확장 |

## 4. 사이트 반영 원칙

- 홈은 브랜드 설명보다 “이 고객에게 어떤 결과를 주는가”를 먼저 보여준다.
- STANDARD / LOCAL은 취향 분기가 아니라 사용 목적 분기로 설명한다.
- 문의 단계 전까지는 가격보다 적용 상상과 납품 가능성을 강화한다.
- 작품 상세는 감상보다 공간 적용 판단에 도움이 돼야 한다.
